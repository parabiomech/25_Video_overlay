# 🎬 Motion Overlay Editor

비디오에서 여러 시점의 움직임을 하나의 이미지/영상으로 합성하는 웹 기반 도구입니다.

## ✨ 주요 기능

- **다중 시점 선택**: 비디오에서 원하는 순간들을 선택
- **자동 객체 추출**: 배경 차이를 이용한 지능형 객체 감지
- **실시간 미리보기**: 추출 결과를 실시간으로 확인하며 조정
- **이중 출력**: 정지 이미지 + 모션 트레일 영상 동시 생성

## 🚀 사용 방법

### 1단계: 시점 선택
1. **비디오 파일 업로드** - 모든 비디오 형식 지원
2. **시점 선택하기**
   - 비디오를 재생하며 원하는 순간에 일시정지
   - "시점 추가" 버튼 클릭 또는 `스페이스바` 누르기
   - 여러 시점을 추가하면 자동으로 시간순 정렬
3. 선택한 시점은 하단에 태그로 표시되며, 클릭하면 해당 시점으로 이동

### 2단계: 객체 추출 설정
1. **감지 민감도 조절**
   - 슬라이더를 조절하여 배경과 객체 구분 정도 설정
   - 낮은 값(10-30): 작은 변화도 감지 (미세한 움직임)
   - 높은 값(50-100): 큰 변화만 감지 (명확한 움직임)
2. **미리보기 확인**
   - 왼쪽: 원본 프레임
   - 오른쪽: 추출된 객체 마스크 (하얀색 = 감지된 객체)
3. 마스크에서 객체가 깨끗하게 보일 때까지 조정

### 3단계: 결과 생성
1. **합성 사진**: 모든 시점의 객체가 하나의 이미지에 오버레이
2. **모션 트레일 영상**: 시간에 따라 객체가 누적되는 영상
3. 각각 다운로드 버튼으로 저장 가능

## 💡 활용 사례

- **스포츠 분석**: 골프/야구 스윙 궤적, 공의 이동 경로
- **컬링**: 스톤의 이동 경로와 최종 위치
- **댄스/안무**: 동작의 연속성과 동선 분석
- **과학 실험**: 물체의 운동 경로 시각화
- **예술 작품**: 시간의 흐름을 담은 창의적 표현

## 🎯 사용 팁

### 최적의 결과를 위한 조건
- ✅ **고정된 카메라**: 삼각대 사용 권장
- ✅ **일정한 조명**: 조명 변화가 적을수록 정확
- ✅ **명확한 대비**: 배경과 객체의 색상 차이가 클수록 좋음
- ✅ **깨끗한 배경**: 단순한 배경일수록 추출 정확도 향상

### 키보드 단축키
- `Space`: 현재 시점을 추가 (1단계에서)

### 문제 해결
- **객체가 제대로 감지되지 않을 때**: 민감도 슬라이더를 낮춰보세요
- **배경까지 감지될 때**: 민감도 슬라이더를 높여보세요
- **일부분만 감지될 때**: 배경과 객체의 색상 차이를 확인하세요

## 🛠️ 기술 스택

- **순수 HTML/CSS/JavaScript** - 별도 설치 없음
- **Canvas API** - 이미지 처리
- **MediaRecorder API** - 영상 생성
- **차분(Difference) 알고리즘** - 객체 추출
